常見問題

常見問題?

  • Q1. M ONE 手機版 可以將樓盤轉為C盤嗎?
  • Q2. 什麼是已剔除
  • Q3. 廚房、浴室、衣帽間是指定標籤嗎?
  • Q4. 為什麼我可以登入MRMS 但入唔到M ONE
  • Q5. 點解我轉完TEAM之後, 用唔到某D手機APP?
  • Q6. 佢話我手機程式未係手機登出, 但我已經入唔返點算?
  • Q7. 放盤易無法搜尋某個樓盤, 于是無法遞交
  • Q8. 如果同事可以入到PULL MAIL, 但無法收發可以點做?
  • Q9. 我M1用得好慢點算?
  • Q10. 點解QR CODE打卡打唔到既

常用回答?

  1. ANS.1 不能, 只可以使用網頁版轉
  2. ANS.2 通常是, 你的樓盤將會出到街網但仍未, 其中一個可能是距離最後更新不足2小時
  3. ANS.3 不是, 所以你很可能未有足夠的指定標籤; 如你更改了其中的標籤種類, 你需要重新選擇相片予主管再審批, 或者你需要刪除所有舊相, 重新選擇具有3種或以上指定標籤的相片, 重新遞交予主管審批
  4. ANS.4 M ONE 可以係密碼入多於8個位, 但密碼DEFAUL左最多8個位
  5. ANS.5 你有無係美聯APPSTORE 都進行了登出, 並使用新TEAM CODE 登入? 如果有, 試下重新安裝D手機APP
  6. ANS.6 你可以係內聯網, 搵返資訊科技部, 管理登入裝置
  7. ANS.7 轉成現盤後, 30分鐘之後才會出現
  8. ANS.8 首先CHECK左佢既WEBMAIL入唔入到先
  9. ANS.9 同事請試下清cookie:開chorme->右上角「...」->「設定」->「清除瀏覽紀錄」
  10. ANS.10 試試以下方法 1.打開開單易 (手機定位要開左) 2.右上角->打卡 3.load ->「請打開定位服務」 4.出手機桌面，開GOOGLE MAP，按定位 5.定位完後回到開單易->按掃瞄QR CODE
